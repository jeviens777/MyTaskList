## 📱 Deskripsi Aplikasi
MyTaskList App adalah aplikasi sederhana berbasis React Native yang digunakan untuk mencatat dan mengelola tugas harian. Aplikasi ini memungkinkan pengguna menambahkan, menyelesaikan, dan menghapus tugas dengan tampilan yang interaktif dan mudah digunakan.

---

## ✅ Fitur Aplikasi
- [x] Menambahkan tugas
- [x] Menandai tugas selesai
- [x] Menghapus tugas
- [x] Statistik tugas (total, belum selesai, selesai)
- [x] Timestamp waktu tugas
